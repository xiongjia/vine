# Map

Interactive map powered by MapLibre GL with OpenStreetMap tiles.

## Default

Centered on Tokyo with default zoom and height.

```tsx preview
<MapView className="rounded-md" />
```

## Custom Center

New York City with custom zoom level.

```tsx preview
<MapView className="rounded-md" center={[-74.006, 40.7128]} zoom={12} />
```

## Inside a Card

```tsx preview
<Card className="w-96">
  <CardHeader>
    <CardTitle>MapLibre GL</CardTitle>
    <CardDescription>With OpenStreetMap tiles</CardDescription>
  </CardHeader>
  <CardContent>
    <MapView className="rounded-md" />
  </CardContent>
</Card>
```

## API

| Prop        | Type               | Default               | Description        |
| ----------- | ------------------ | --------------------- | ------------------ |
| `center`    | `[number, number]` | `[139.6917, 35.6895]` | Map center (Tokyo) |
| `zoom`      | `number`           | `10`                  | Zoom level         |
| `height`    | `string`           | `"300px"`             | Map height         |
| `className` | `string`           | —                     | CSS class          |
| `style`     | `CSSProperties`    | —                     | Inline styles      |
